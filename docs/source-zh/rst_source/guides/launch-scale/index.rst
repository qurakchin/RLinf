启动与扩展
==========

当你需要在单机默认运行之外启动任务，或将 RLinf 接入真实硬件时，使用这些指南。

.. list-table::
   :header-rows: 1
   :widths: 32 68

   * - 指南
     - 内容
   * - :doc:`多节点训练 <../multi_node>`
     - 在多个节点上启动训练。
   * - :doc:`异构集群 <../hetero>`
     - 配置混合硬件与节点分组。
   * - :doc:`云边协同 <../cloud_edge>`
     - 在云端与边缘之间拆分推理与训练。
   * - :doc:`真机机器人 <../realworld_robot>`
     - 在真实机器人硬件上运行 RL。
